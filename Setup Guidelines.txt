How do start my project.
(STEP 1) In terminal, type cd Server(or simply go to Server folder) and type "npm start" to start the server.
(STEP 2) Open a new terminal, type cd Bookapp(or simply go to Bookapp folder) and type "npm run dev" to launch the Vite+React app.
(STEP 3) CTRL+Click on the link which will appear in terminal.
